# Stage 0.1 MVP Scope

最后更新：2026-03-22

## 产品定位

BookLearning 的第一版定位为：

- 自用优先的 AI 辅助 PDF 学习工具
- 面向“逐页阅读、逐页讲解、逐步积累结果”的工作流
- 不以通用 RAG 文档问答平台为目标

## 明确纳入第一版的能力

### 阅读与任务流

- 打开本地 PDF 文件
- 预览 PDF、显示当前页与总页数
- 指定当前页、单页或页范围发起解析
- 支持逐页排队执行，不强制一次处理整本书

### AI 讲解

- 支持编辑全局提示词
- 支持调用视觉模型处理单页图像
- 支持保存页级讲解结果
- 支持单页重跑
- 支持在阅读界面右侧展示讲解

### 配置与保存

- 支持配置 `base URL`、`API key`、模型名等基础项
- 支持记录多个配置并切换
- 支持把项目状态、提示词、页级讲解结果保存到本地

### 导出

- 第一版预留导出链路
- 导出优先级：项目数据导出 > HTML 导出 > PDF 双栏导出

## 第一版明确不优先实现的内容

- OCR 识别与扫描件深度支持
- 通用知识库/RAG/向量数据库
- 多用户协作
- 云端账号体系
- 商业化计费相关能力

## 核心工作流

1. 用户打开本地 PDF
2. 用户编辑全局提示词
3. 用户选择当前页或页范围
4. 系统逐页生成页面图像并调用视觉模型
5. 系统把讲解结果与页码绑定保存
6. 用户在左侧看原文、右侧看讲解
7. 用户可对某一页重跑或手动修订
8. 后续把结果导出为学习版文档

## 成功标准

只要第一版满足以下条件，就算达标：

- 可以稳定打开本地 PDF
- 可以对指定页发起 AI 讲解
- 可以按页保存和回显讲解
- 可以单页重跑而不影响整本书
- 可以长期在本地重复使用
